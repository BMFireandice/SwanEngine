About This Project


This research search engine was created as a school junior project.

Scope of This Project

The scope of this project was to create a search engine primarily for research purposes. This search engine will have a searching function based on Google Scholarly's website database. It will return the same results as Google Scholarly, but it will be displayed in a more comfortable manner.

Contraints/Limitations

There were major limitations while I was creating this search engine. The first one is about captcha. Google Scholarly implements a captcha service in their engine, so when a user uses mine, they will have to go through the same captcha. However, with the way I implemented the engine, I couldn't display the captcha for the user to solve, so I had to code a way to automatically solve them. I ended up using a paid service to solve the captchas automatically, so users can search without needing to solve them.

Another limitation is time. I started working on the project around the start of March, and the project is due in May, so I have around two months in total to work on it. During this time, a lot of the focus was spent on troubleshooting and testing the searching aspect, and I had no time to work on everything that I wanted to. The VPN and proxy service that I initally wanted to implement could not be implemented in the sake of time. The settings feature is also very limited, as I did not have the time to make a larger settings menu.

Finally, I could not end up making the search feature toggleable. I initally wanted to allow the user to use this engine as both a normal search engine and as a research search engine, and while I have some ideas on how to implement this feature, I do not have the time to implement it. As a result, I only made this search engine for research purposes, as that is the primary purpose of the engine.
